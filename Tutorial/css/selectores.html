Selectores

si tienes un h1, o lo que sea y en css pones h1{color: black;}. se cambiaran todos los h1. en cambio puedes hacer div, o p con una id para que solo se cambien los especificos. (SELECTORES)
No puedes poner mismo id a varios elementos, pero si poner clases pero en vez de #, pones .

HTML

p id="nombre" TEXTO /p


CSS

p#nombre{
color: red;
}
