Posiciones

Display

para efectuarse tiene que estar en position: static;
Display:

  1. inline-block --> Ocupan lo que tienen que ocupar al lado de otros siempre que hay espacio
  2. inline --> Se posicionan al lado de otro pero se comportan como un solo bloque
  3. none --> se pone fantasma

Coordenadas

Tiene que estar en position: relative;
a donde se deberia pintar lo que deseas, puedes poner:

  1. top: x
  2. left: x
  3. right: x
Cuando pones position: absolute; el objeto pasa a ser intocable, todo se pone encima suyo, como si no estubiera.

Position

  1. Static --> Se posiciona respecto al padre, al inicio. no se mueve.
  2. Relative --> Cuando varies la posicion, se cambiara respecto al padre.
  3. Absolute --> el objeto pasa a ser intocable, todo se pone encima suyo, como si no estubiera.
  4. Fixed --> Se posiciona la caja en el body, siempre. cuando bajas la pantalla se queda fijado.(menus)
