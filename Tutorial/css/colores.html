Colores

Puedes poner el color de varias formas:

  1. Poner el nombre del color (visitar pagina web para saberlos)
  2. Poner el RGB de tal forma: rgb(x,y,z)
  3. Poner el RGB hexadecimla de tal forma: #xxxxxx
  4. Poner el RGB porcentual de tal forma: rgb(x%,y%,z%)
  5. Poner el HSL de tal forma: hsl(x,y%,z%)
  6. Poner el RGB con transparencia de tal forma: rgba(x,y,z,t(menor que 1))
